üì• Data Flow (Request)

  1. 1 User interacts with dashboard (filters, search, clicks pipeline stage)
  2. 2 Frontend sends HTTP request to API Gateway (e.g., GET /api/bills?status=committee)
  3. 3 Express server routes request to appropriate handler
  4. 4 Backend calls Open States API with jurisdiction=ok and filters
  5. 5 Process and enrich data (determine bill stages, format dates)
  6. 6 Return JSON response to frontend with enhanced bill data
  7. 7 Dashboard renders updated UI with bills and visualizations
